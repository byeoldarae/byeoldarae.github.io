생활코딩 WEB2 CSS-4. CSS의 기본문법

정리A. 웹페이지에 css삽입하는 방법

  1. style 태그의 사용
    style a {color:red;} /style
    a {} : 누구에게 적용할건지의 정보= 선택자(selector)
    color:red; : 효과 (declaration)
    color: 속성(property)
    red; value
    ;(semi-clone): 효과들의 구분선
  2. style 속성의 사용
    style="color:red"
    style="": html의 속성
    color:red : 값(반드시 css의 효과가 들어온다)
    이 경우 style 속성이 위치한 곳에 효과를 주어 위치지정이 따로 필요 없다.
