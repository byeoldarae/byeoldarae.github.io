생활코딩 WEB-2 Javascript-2. 수업의 목적

Javascript: 사용자와 상호작용
html은 한번 출력되면 변하지 않는다.
onclick이라는 속성뒤에는 자바스크립트가 와야한다.
Javascript은 html을 제어하는 언어이다.

생활코딩 WEB-2 Javascript-3.HTML과 JS의 만남: script 태그

html은 script 태그 안쪽에 오는것을 javascript로 해석한다.
: html에게 자바스크립트라는것을 알려주는 태그
document.write(1+1);
화면에는 1+1의 값인 2가 출력된다.

document.write: 자바스크립의 코드, 웹페이지에 어떤 글씨를 출력할 때 쓰는코드

생활코딩 WEB-2 Javascript-4.HTML과 JS의 만남: 이벤트


ONCLICK 속성값으로 반드시 JAVASCRIPT가 와야한다.
ONCLICK 속성의 속성값은 웹브라우저가 기억하고 있다가 ONCLICK 속성이 위치하고 있는 태그에 사용자가 클릭했을때,
기억하고있었던 Javascript의 코드를 Javascript의 문법에 맞게 해석해서 웹브라우저가 동작할것이다.


10~20가지의 이벤트로 사용자와 상호작용이 가능하다.
onclick, onchange, onkeydown 등을 이벤트라 부른다.
